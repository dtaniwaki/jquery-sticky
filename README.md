# jQuery Sticky

jQuery Sticky is a jQuery plugin that gives you the ability to make any element on your page stay sticky in the containers.

## Usage

```html
<script src="jquery.js"></script>
<script src="jquery.sticky.js"></script>
<script>
  $(function(){
    $("#sticky").sticky();
  });
</script>
```
